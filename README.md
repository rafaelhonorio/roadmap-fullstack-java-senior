# Roadmap Fullstack Java Senior

Este repositório registra minha evolução técnica para me tornar um Desenvolvedor Fullstack Java Senior/Especialista.

## Objetivo

Construir uma base sólida e progressiva em:

* lógica de programação;
* Java;
* orientação a objetos;
* banco de dados;
* SQL;
* Spring Boot;
* APIs REST;
* testes;
* TypeScript;
* Angular;
* segurança;
* Docker;
* CI/CD;
* arquitetura;
* performance;
* troubleshooting;
* boas práticas de engenharia de software.

## Método

Cada fase possui:

* conceitos estudados;
* exercícios práticos;
* projeto relacionado;
* anotações;
* evidências;
* retrospectiva.

Toda semana deve gerar pelo menos uma evidência técnica versionada.
